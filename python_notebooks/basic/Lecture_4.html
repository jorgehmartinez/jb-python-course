4. If and Loops#

4.1. If condition#

Function if test conditions.

  • Equals: a == b

  • Not Equals: a != b

  • Less than: a < b

  • Less than or equal to: a <= b

  • Greater than: a > b

  • Greater than or equal to: a >= b

4.1.1. The structure of the code#

if test expresion:
    Code to excute

4.1.2. If condition with more than 1 expression#

We will use elif. This function allows us to add more test expressions.

if test expresion1:
    Code1

elif test expresion2:
    Code2

elif test expresion3:
    Code3

elif test expresion4:
    Code4

elif test expresion5:
    Code5

else :
    Code6

Python will read the conditions of the test in order. If Test expression 1 is True, the rest of the test expressions will not be evaluated. R execute Code2 and will not test the next conditions.

In case no Test expression is True, the Code6 will be excuted.

Difference between == and is operator in Python#

  1. == operator compares the values of both the operands and checks for value equality.

  2. is operator checks whether both the operands refer to the same object or not (present in the same memory location).
